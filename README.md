# 🚀 Magnet Bridge - Chrome Extension

Magnet Bridge is a powerful Chrome extension that extracts and lists torrent magnet links from any webpage. It allows users to easily copy magnet links, download .torrent files, and integrate with cloud torrent services like **Seedr, Bitport, Put.io, and Torbox**.

---

## 🌟 Features

✅ **Extract Magnet Links** - Automatically find and list all magnet links on a webpage.  
✅ **Copy Magnet Links** - Quickly copy any magnet link with one click.  
✅ **Download .torrent Files** - Download .torrent files directly if available.  
✅ **Cloud Torrent Integration** - Configure API keys to send torrents directly to Seedr, Bitport, Put.io, or Torbox.  
✅ **User-friendly Popup** - A clean and simple interface for managing your torrents.  

---

## 📥 Installation

1. **Download or Clone** this repository.
2. Open **Chrome Extensions** page (`chrome://extensions/`).
3. Enable **Developer mode** (top right corner).
4. Click **Load unpacked** and select the extension folder.
5. Enjoy using Magnet Bridge! 🎉

---

## 🔧 How to Use

1. Open a webpage containing torrent magnet links.
2. Click the **Magnet Bridge** extension icon.
3. View all extracted magnet links in the popup.
4. Copy, download, or send torrents to your preferred cloud service. 🚀

---

## 🔑 API Key Setup (Cloud Torrent Services)

To integrate with **Seedr, Bitport, Put.io, or Torbox**:

1. Go to the extension **Settings**.
2. Enter your API key or access token.
3. Save the settings and start adding torrents to your cloud storage. ☁️

---

## 📜 Permissions

This extension requires the following permissions:
- **Active Tab** - To scan and extract magnet links.
- **Storage** - To securely store API keys (using Chrome Sync Storage).

🔒 **Your data is stored securely on your local system and synced across devices if Chrome Sync is enabled.**  
[Learn more about Chrome Sync Storage](https://developer.chrome.com/docs/extensions/reference/storage#property-sync)
